dotfiles
========

### Vim Binds(.vimrc)

	F2	- Save(:w)
	F5	- New tab
	F6	- Prev tab
	F7	- Next tab
	F11	- Save & exit (:wq!)
	ESC	- Exit without save (:q!)
	CtrlP	- Use [ctrlp.vim](http://kien.github.io/ctrlp.vim/)

### Terminal

	.bashrc		- *nix terminal configuration
	.gitconfig	- Aliases for git
